# Fastify template - Typescript


## Installation

```bash
npm install foobar
```

## Environments

```javascript
MONGO_USER=
MONGO_PASS=
MONGO_HOST=
MONGO_PORT=
MONGO_DATABASE=
PORT=
NODE_ENV=
JWT_DEV=
```

## Routes
/user (POST)          - Create a new user
/user (GET)           - Fetch user information using JWT
/login (POST)         - Create JWT token using Basic Auth
/documentation (GET)  - Get OpenAPI doc
